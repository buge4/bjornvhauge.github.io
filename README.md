## Deployment

This site is deployed on GitHub Pages with custom domain support.

### Custom Domain Setup

1. Add CNAME record pointing to `<username>.github.io`
2. Enable HTTPS in GitHub Pages settings
3. Wait for DNS propagation (up to 24 hours)
